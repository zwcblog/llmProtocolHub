# LLM Protocol Hub

一个面向 AI 厂商与协议的 **官方入口导航站**。

它不试图在仓库里重复维护一整套详细技术文档，而是把最重要的官方入口整理清楚：

- 官网
- 官方文档
- API Reference
- SDK / GitHub
- Changelog / Release Notes
- 控制台或平台入口

目标很简单：

> 让你在最短时间内，找到最准确的官方信息。

## 站点定位

这个项目更接近：

- 官方文档索引
- AI 厂商入口导航
- 协议规范入口集合

而不是：

- 二次改写的 API 教程
- 大而全的协议百科
- 长篇参数说明书

具体细节、字段和最新变更，以各自官网为准。
